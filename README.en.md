# 基于javaweb网络教学系统

#### Description
管理员：对全局的管理，比如用户，老师，题目，留言板，发布资料等。
老师：可以实现在线发布题目，下载学生作业等。
学生：可以浏览资料，公告，发表留言等。总之完整的实现了一个网络教学平台的功能。
用了技术框架： HTML+CSS+JavaScript+jsp+mysql
